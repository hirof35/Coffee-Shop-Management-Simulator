琥珀色の経営学 - Amber Management -
落ち着いたJAZZが流れる空間で、コーヒーショップを経営するクリッカー＆リソース管理シミュレーションゲーム。

☕️ ゲーム概要
「琥珀色の経営学」は、キッチンカーからスタートし、資産を築いて大型店舗へと拡大していく本格的な経営シミュレーションです。
単なるクリッカーゲームではなく、「在庫管理」「体力（ワークライフバランス）」「マーケティング」「固定費」の要素が絡み合う、戦略的な判断が求められる設計になっています。

🌟 主な機能
リアルタイム経営: お店を開けている間、知名度に応じて定期的にお客が来店します。

リソース管理: 豆や牛乳の在庫を切らさないよう、卸売なども利用して効率的に仕入れを行います。

スタッフ雇用: バリスタを雇うことで、コーヒーの抽出を自動化（自動クリッカー要素）できます。

店舗拡大: キッチンカーから路面店、さらには大型店舗へ。ランクアップで雇用枠も拡大しますが、家賃などの固定費リスクも発生します。

実績システム: 「期待の新人」「琥珀色の極致」など、やり込み要素としての称号と報酬システムを搭載。

セーブ機能: LocalStorageを利用したセーブ/ロードに対応。

🎮 遊び方
調理: 最初は自分でコーヒーを淹れて在庫を作ります。

営業: 「お店を休む」ボタンを切り替えて営業を開始。お客が来ると在庫が売れて資金が増えます。

仕入れ: 資金を使って豆や牛乳を補充します。

投資: 広告を出して知名度を上げたり、バリスタを雇って自動化を進めたりします。

休憩: 体力がなくなると過労で倒れてしまいます。適度に店を閉めて休息を取りましょう。

🛠 技術スタック
Frontend: HTML5, CSS3 (CSS Variables for Theming)

Logic: Vanilla JavaScript

Storage: LocalStorage API

Audio: HTML5 Audio API (BGM: Jazz Curiosity)

🚀 実行方法
特別なサーバー設定は不要です。

リポジトリをクローンまたはファイルをダウンロードします。

index.html（または該当のファイル）をブラウザで開きます。

画面をクリックするとBGMが再生され、経営が始まります。
